Pastel accents — semantic mapping

Always pair pastel backgrounds with text-ink. Each pastel has a job — don't freestyle.

pastel-pink = primary#f5a8d8 · primary action, liked, ring
pastel-yellow#f8e47a · attention, active tab/filter
pastel-mint = success#a8e6b0 · in library, completed
pastel-sky = info#a8d8f0 · downloading, in progress
pastel-lavender#c9b8f0 · secondary accent only
pastel-red = destructive#f0978b · error, delete, failed

Radius & spacing

Base --radius: 1rem. Computed Tailwind values: rounded-md 0.8rem, rounded-lg 1rem, rounded-xl 1.4rem, rounded-2xl 1.8rem. Core principle: roundness scales down as elements get smaller.

Do's & don'ts

The flat pastel system in one list.

DOKeep it flat — no shadows, gradients, or glows. Depth = surface scale (background → surface → surface-2 → accent).
DOAlways text-ink on pastel backgrounds. Never white/foreground.
DOFollow the radius ladder: 2xl modals/cards → xl tiles/inputs/menus/rows → lg row thumbs → md tiny thumbs → full for pills & artists. Roundness decreases as elements shrink.
DOUse shared components: Button, Badge, Card, Input — and extract IconButton, TrackRow, StatusPill, Menu, Cover instead of re-rolling them per page.
DOOne icon per action from the canonical map, h-4 w-4 by default, lucide only.
DOSemantic pastels: pink primary/liked · mint success/in-library · sky info/downloading · yellow attention/active tab · red error/destructive · lavender secondary only.
DOHover language: pastel/80 (filled pastels) · bg-surface-2 + foreground (ghost/icon buttons) · bg-accent (secondary) · pastel swap (danger hover → bg-pastel-red text-ink).
DON'TIntroduce new hex values — extend :root tokens first.
DON'TUse divide-y track lists, bordered row play buttons, rounded-md tiles, font-semibold titles, or underline tabs — all are legacy patterns to migrate.
DON'TUse border-border (pure ink, invisible) for separators — use border-foreground/10. border-2 border-transparent is only for layout-stability on interactive chips.
DON'TMix size-* and h-* w-* icon sizing — standardize on h-* w-*.
